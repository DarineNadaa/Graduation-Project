The API Loop

Four steps. One score.
Total clarity.

01
SIEM / EDR fires

Your existing stack generates an alert. Nothing changes in your pipeline or tooling.

02
ATTENSE ingests

A single POST to the API with an event payload. Sandboxed — never touches production.

03
AI engine scores

Correlates behavior against MITRE ATT&CK. Calculates TTD, TTC, and anomaly signals.

04
Report delivered

SHA-256-hashed PDF + JSON. Immutable record. Longitudinal trend data appended automatically.

attense_integration.sh 
# Three lines. That's the integration.

curl -X POST https://api.attense.io/v1/incidents \
  -H "Authorization: Bearer at_live_<YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "incident_id":  "INC-2026-0391",
    "siem_source":  "splunk",
    "event_type":   "lateral_movement",
    "severity":     "high",
    "detected_at":  "2026-06-07T14:23:07Z",
    "contained_at": "2026-06-07T14:26:03Z",
    "analyst_id":   "usr_snr_042"
  }'

# Response → TTD, TTC, AI score, MITRE tags, SHA-256 audit hash
